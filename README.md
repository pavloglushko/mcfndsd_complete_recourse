Multi-commodity flow network design problems with demand uncertainty and complete recourse.



Based on the R instances from Rahmaniani et al. “A Benders Decomposition Method for Two-Stage Stochastic Network Design Problems.” CIRRELT 22 (2017): 2–3.

Complete recourse version is generated by adding slack variables to the right-hand side of the flow conservation
constraints.

Format: smps (three files)


Key dimensions of the instances are listed in the table below, 
including the number of binary variables on the first stage 1st Bin, 
the number of continuous variables on the second stage per scenario, 2nd Cont, 
and the number of constraints (rows) on the second stage, 2nd Constraints. 
The dimensions of the second stage are provided per scenario.

Selected instances were solved on a workstation with AMD Ryzen 9 3950X 16-Core (32 Threads) 3.5 GHz processor working on a 64-bit Windows 10 operating system. 
We use IBM ILOG Cplex MIP solver with default parameters and the solution time limit set to three hours per instance.
If the instance is not solvable within that time, we report the solution time value as 10800 and the objective value as "TIME_LIMIT".
